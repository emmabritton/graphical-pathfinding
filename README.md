## Graphic Pathfinding

(Only has A* at the moment)

#### Keys

* **ESC** or **Q** will close the program
* Map:
  * **Up, Down, Left, Right** to highlight map
  * **Enter/Return** to select
* Algorithms:
  * **0, 1** select
* Runner:
  * **[** faster
  * **]** slower
  * **p** toggle manual mode
  * **space** advance one tick in manual mode
